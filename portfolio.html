Pace Booking Application Utilising NFC technologies


The Pace App is a secure event booking application that implements NFC technology as an added feature. Coded in Java, the application is based on Android devices using Android Studio as the IDE. With the use of Google's Firebase as the database for user data, this allows secure access to the database directly from the client-side code.
